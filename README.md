# Tea Origin Viewer

茶葉産地管理システム - 茶葉の産地情報を地図上で管理・表示するWebアプリケーション

## 概要

Tea Origin Viewerは、茶葉の産地情報を効率的に管理し、地図上で視覚的に確認できるシステムです。茶葉の調達元、ロット情報、収穫日などの詳細情報を一元管理し、産地の地理的分布を把握できます。

## 機能

### 🌍 地図表示機能
- **Leaflet.js**を使用したインタラクティブマップ
- 茶葉産地の位置をマーカーで表示
- マーカークリックで詳細情報をポップアップ表示
- 静岡県牧之原台地をデフォルト座標に設定

### 📝 茶葉登録機能
- 新規茶葉の産地情報を登録
- トグル式の登録フォーム（開閉可能）
- 以下の情報を管理：
  - 原料名
  - 調達元（農園名）
  - 緯度・経度
  - ロットID
  - 収穫日
  - 品質メモ

### 📋 ロット一覧表示
- 登録済み茶葉の一覧表示
- テーブル形式での詳細情報表示
- ページネーション機能（1ページ5件）
- 縦線・横線で区切られた見やすいレイアウト

### 🔔 通知機能
- 登録成功時のトースト通知
- 自動非表示機能（3秒後）

## 技術スタック

### バックエンド
- **Ruby on Rails 8.0.2** - Webフレームワーク
- **SQLite3** - データベース
- **Kaminari** - ページネーション

### フロントエンド
- **Stimulus.js** - JavaScriptフレームワーク
- **Leaflet.js** - 地図ライブラリ
- **Propshaft** - アセットパイプライン

### デザイン
- **レスポンシブデザイン** - モバイル対応
- **モダンUI** - グラデーション、カードデザイン
- **カスタムCSS** - Tailwind CSS不使用

## セットアップ

### 前提条件
- Ruby 3.0以上
- Node.js 18以上
- Git

### インストール手順

1. リポジトリをクローン
```bash
git clone https://github.com/io0323/tea-origin-viewer.git
cd tea-origin-viewer
```

2. 依存関係をインストール
```bash
bundle install
npm install
```

3. データベースをセットアップ
```bash
bin/rails db:create
bin/rails db:migrate
```

4. サーバーを起動
```bash
bin/rails server
```

5. ブラウザでアクセス
```
http://localhost:3000
```

## 使用方法

### 茶葉の登録
1. 「新規茶葉登録」セクションをクリックしてフォームを開く
2. 必要な情報を入力
3. 「登録」ボタンをクリック
4. 登録成功の通知が表示される

### 地図の確認
- 登録した茶葉の産地が地図上にマーカーで表示される
- マーカーをクリックすると詳細情報が表示される

### ロット一覧の確認
- 登録済みロット一覧で全茶葉の情報を確認
- ページネーションで複数ページを閲覧

## データベース構造

### TeaOrigin モデル
- `name` - 原料名
- `location_name` - 調達元（農園名）
- `latitude` - 緯度
- `longitude` - 経度
- `lot_id` - ロットID
- `harvested_on` - 収穫日
- `notes` - 品質メモ
